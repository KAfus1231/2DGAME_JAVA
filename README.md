# 2DGAME_JAVA

Небольшая 2D-игра на Java, реализованная с использованием NetBeans/IntelliJ IDEA.

##  Что внутри
- `src/` — исходный код
- `res/` — игровые ресурсы (графика, звук и т.д.)
- `.idea/`, `*.iml` — конфигурации IDE (можно игнорировать или убрать)
